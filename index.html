CommodityTracking

Skeleton Tracking using a single, inexpensive webcam.

Today, digital cameras are commodities.

Most consumer laptops have a webcam built-in.

Traditional solutions require expensive, special equipment

CommodityTracking enables anyone with a webcam to experience natural human interaction without special sensors.

Building and Installation

To build CommodityTracking, install OpenCV.
Clone the repository on Github.
Run make.
The standard sample can be run with ./demo.
